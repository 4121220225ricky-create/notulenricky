📖 Cara Penggunaan & Info Groq Free Tier

          

            Setup Groq (sekali, gratis):

            1. Buka console.groq.com/keys

            2. Login/daftar (gratis, tanpa kartu kredit)

            3. Klik "Create API Key" → salin key-nya

            4. Tempel di kolom "Groq API Key" di atas → Simpan



            Batas free tier Groq (per hari):

            • 2.000 request transkripsi/hari

            • Cukup untuk ~11 jam audio per hari

            • Jika lebih, lanjutkan keesokan harinya



            Estimasi waktu proses:

            • 1 jam audio ≈ 180 segmen × ~2 detik = ~6 menit

            • 7 rekaman × 1.5 jam ≈ 60–90 menit total

            • Bisa dibagi 2 hari jika terkena rate limit



            Tip jika kena rate limit (error 429):

            Website akan otomatis tunggu 60 detik lalu lanjut. Jangan tutup browser.
