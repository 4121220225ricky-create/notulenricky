📖 Cara Penggunaan

          

            Langkah 1 — Setup API Key:

            Masukkan OpenAI API Key (wajib untuk transkripsi). Anthropic key opsional untuk notulensi otomatis.



            Langkah 2 — Upload Audio:

            Upload satu atau lebih file audio dari sidebar kiri. Tidak ada batas ukuran atau durasi.



            Langkah 3 — Transkripsi:

            Klik "Mulai Transkripsi". File panjang otomatis dipotong per segmen ~55 detik, lalu ditranskripsi berurutan dan digabung jadi satu teks utuh.



            Langkah 4 — Notulensi:

            Isi info rapat (opsional), lalu klik "Buat Notulensi" di tab Notulensi. Claude akan mengubah transkrip menjadi notulensi profesional.



            Perkiraan Waktu & Biaya:

            Rapat 1 jam ≈ 65 segmen × ~3 detik = ~3-5 menit proses.

            Biaya Whisper: 1 jam audio ≈ $0.36. Notulensi Claude: ~$0.02/rapat.
